# EXPERIMENT 4: Blog Website with MongoDB CRUD Operations

## 📋 **EXPERIMENT DETAILS**
- **Subject**: Backend Development Lab
- **Experiment No.**: 4
- **Website Name**: Blogily
- **Technology Stack**: Node.js, Express.js, MongoDB, EJS, CSS3

---

## 🎯 **TASK TO BE DONE**

### **Primary Objective**
Develop a complete blog website with full CRUD (Create, Read, Update, Delete) operations using MongoDB as the database and implement user authentication system.

### **Specific Tasks**

#### **1. Database Design & Setup**
- [ ] Design MongoDB schema for Users and Blogs
- [ ] Implement database connection using Mongoose
- [ ] Create User schema with authentication fields
- [ ] Create Blog schema with author relationship

#### **2. User Authentication System**
- [ ] Implement user registration (Sign Up) functionality
- [ ] Implement user login functionality
- [ ] Add password hashing using bcryptjs
- [ ] Implement session management
- [ ] Add logout functionality

#### **3. Blog Management System (CRUD Operations)**
- [ ] **CREATE**: Implement blog creation functionality
- [ ] **READ**: Display all blogs on homepage with pagination
- [ ] **UPDATE**: Implement blog editing functionality
- [ ] **DELETE**: Implement blog deletion functionality
- [ ] Add authorization (users can only edit/delete their own blogs)

#### **4. User Interface Development**
- [ ] Design responsive navigation bar
- [ ] Create attractive homepage with blog grid layout
- [ ] Design login and signup forms
- [ ] Create blog creation and editing forms
- [ ] Implement search functionality
- [ ] Add modern CSS styling with gradients and animations

#### **5. Additional Features**
- [ ] Implement search functionality across blog titles, content, and authors
- [ ] Add confirmation dialogs for delete operations
- [ ] Implement responsive design for mobile devices
- [ ] Add user-friendly error handling and validation

## 🚀 **SETUP INSTRUCTIONS**

### **Prerequisites**
1. Node.js installed on system
2. MongoDB running locally or MongoDB Atlas account
3. Code editor (VS Code recommended)
4. Git for version control

### **Installation Steps**
1. Clone/download the project files
2. Run `npm install` to install dependencies
3. Start MongoDB service
4. Run `npm start` to start the server
5. Open `http://localhost:3000` in browser
